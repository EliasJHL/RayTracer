Calculates how a ray scatters when hitting this material.

This method determines whether a ray is absorbed or scattered by the material and if scattered, in what direction and with what attenuation.

Parameters
rayInThe incoming ray that hits the surface
recThe hit record containing information about the intersection
attenuationOutput parameter for the ray's color attenuation
scatteredOutput parameter for the scattered ray
Returns
bool True if the ray was scattered, false if it was absorbed
